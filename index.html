Panduan Upload ke GitHub (Sekali Setup)
1
Daftar & buat repo GitHub
Pergi ke github.com → daftar gratis → klik New repository → beri nama aturkuy-releases → pilih Public → Create.
2
Upload APK via GitHub Releases
Di repo → klik Releases → Create a new release → isi tag (contoh: v1.2) → upload file APK di bagian "Attach binaries" → klik Publish release. Setelah itu klik kanan APK → Copy link address → paste ke form di atas.
3
Upload latest-version.json ke repo
Download file JSON dari tombol di atas → di repo GitHub klik Add file → Upload files → upload latest-version.json → Commit changes.
4
Aktifkan GitHub Pages
Settings repo → Pages → Source: Deploy from a branch → Branch: main, folder: / (root) → Save. Tunggu ~2 menit, URL kamu akan jadi:
https://usernamekamu.github.io/aturkuy-releases/latest-version.json
5
Masukkan URL ke kode app
Di index.html cari baris:
const OTA_CHECK_URL='https://your-server.com/...'
Ganti dengan URL dari langkah 4 di atas. Build ulang APK → selesai! Untuk update berikutnya cukup ulangi langkah 2–3.
Generator URL GitHub Pages

Masukkan username GitHub kamu untuk generate URL yang siap dipakai di kode app.
