Upload APK ke Netlify Drop

APK tidak bisa diupload langsung dari browser ke GitHub (CORS restriction). Pakai Netlify Drop — gratis, drag & drop, langsung dapat URL permanen dalam 30 detik.

1
Buka app.netlify.com/drop di tab baru
2
Buat folder baru di HP/PC, taruh file APK di dalamnya, lalu drag & drop folder itu ke halaman Netlify Drop (harus folder, bukan file langsung)
3
Netlify akan beri URL seperti https://xxxxx.netlify.app/nama-file.apk — copy URL lengkap itu
4
Paste URL APK di field bawah, lalu isi form dan klik Publish
